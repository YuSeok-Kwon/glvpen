# 세이버메트릭스 (Sabermetrics) 시스템

**최종 업데이트**: 2026-03-13
**대상 시즌**: 2023 ~ 2025 (다시즌)
**데이터 소스**: Statiz 크롤링 기본 지표 → 파생 계산

---

## 개요

KBO 선수의 기본 기록(H, AB, PA, IP 등)으로부터 **파생 지표**를 산출하여 DB에 적재하는 배치 시스템이다.
크롤링으로 수집된 원시 데이터를 기반으로, 선수의 실제 실력을 보다 정확하게 평가할 수 있는 고급 분석 지표를 제공한다.

---

## 타자 세이버메트릭스 (7개 지표)

### BABIP (Batting Average on Balls In Play)

| 항목 | 내용 |
|------|------|
| **수식** | `(H - HR) / (AB - SO - HR + SF)` |
| **의미** | 인플레이 타구가 안타가 된 비율. 운과 수비의 영향을 평가 |
| **해석** | KBO 평균 약 .300. 극단적으로 높거나 낮으면 평균 회귀 가능성 |
| **저장** | category = `BABIP`, 소수 3자리 |

### ISO (Isolated Power)

| 항목 | 내용 |
|------|------|
| **수식** | `SLG - AVG` |
| **의미** | 순수 장타력. 타율을 제외한 장타율로 순수 파워를 측정 |
| **해석** | .200 이상이면 우수한 장타력 |
| **저장** | category = `ISO`, 소수 3자리 |

### K% (삼진율)

| 항목 | 내용 |
|------|------|
| **수식** | `(SO / PA) × 100` |
| **의미** | 전체 타석 대비 삼진 비율 |
| **해석** | 낮을수록 컨택 능력 우수. KBO 평균 약 20% |
| **저장** | category = `K%`, 백분율 소수 1자리 |

### BB% (볼넷율)

| 항목 | 내용 |
|------|------|
| **수식** | `(BB / PA) × 100` |
| **의미** | 전체 타석 대비 볼넷 비율. 선구안 지표 |
| **해석** | 높을수록 선구안 우수. KBO 평균 약 8~9% |
| **저장** | category = `BB%`, 백분율 소수 1자리 |

### wOBA (Weighted On-Base Average)

| 항목 | 내용 |
|------|------|
| **수식** | `(0.69×BB + 0.72×HBP + 0.89×1B + 1.27×2B + 1.62×3B + 2.10×HR) / (AB + BB + SF + HBP)` |
| **의미** | 출루 이벤트에 가중치를 부여한 종합 타격 지표 (FanGraphs 기본 가중치) |
| **해석** | .320 이상 우수, .370 이상 엘리트 |
| **저장** | category = `wOBA`, 소수 3자리 |
| **참고** | 1B(단타) = H - 2B - 3B - HR 로 역산 |

### AB/HR (타수당 홈런 빈도)

| 항목 | 내용 |
|------|------|
| **수식** | `AB / HR` |
| **의미** | 홈런 1개를 치는데 필요한 타수. 낮을수록 장타력 우수 |
| **해석** | 15 이하면 엘리트 파워히터 |
| **저장** | category = `AB/HR`, 소수 1자리 |
| **조건** | HR > 0인 선수만 계산 (HR = 0이면 저장하지 않음) |

### PA/K (타석당 삼진 빈도)

| 항목 | 내용 |
|------|------|
| **수식** | `PA / SO` |
| **의미** | 삼진 1개당 소요 타석 수. 높을수록 컨택 능력 우수 |
| **해석** | 10 이상이면 뛰어난 컨택 능력 |
| **저장** | category = `PA/K`, 소수 1자리 |
| **조건** | SO > 0인 선수만 계산 |

## 투수 세이버메트릭스 (4개 지표)

### K/9 (9이닝당 삼진)

| 항목 | 내용 |
|------|------|
| **수식** | `(SO / IP) × 9` |
| **의미** | 9이닝 기준 삼진 수. 탈삼진 능력 평가 |
| **해석** | 9.0 이상이면 삼진 능력 우수 |
| **저장** | category = `K/9`, 소수 2자리 |

### BB/9 (9이닝당 볼넷)

| 항목 | 내용 |
|------|------|
| **수식** | `(BB / IP) × 9` |
| **의미** | 9이닝 기준 볼넷 수. 제구력 평가 |
| **해석** | 3.0 이하면 제구력 우수 |
| **저장** | category = `BB/9`, 소수 2자리 |

### HR/9 (9이닝당 피홈런)

| 항목 | 내용 |
|------|------|
| **수식** | `(HR / IP) × 9` |
| **의미** | 9이닝 기준 피홈런 수. 피장타 허용 빈도 |
| **해석** | 1.0 이하면 양호 |
| **저장** | category = `HR/9`, 소수 2자리 |

### LOB% (잔루율)

| 항목 | 내용 |
|------|------|
| **수식** | `(H + BB + HBP - R) / (H + BB + HBP - 1.4 × HR)` |
| **의미** | 출루 허용 주자 중 득점을 허용하지 않은 비율. 위기 관리 능력 |
| **해석** | 70% 이상이면 위기관리 우수, 리그 평균 약 70~72% |
| **저장** | category = `LOB%`, 백분율 소수 1자리 |
| **조건** | LOB% > 0인 경우만 저장, 0~100% 범위로 클램프 |

## 저장 방식

세이버메트릭스 파생 지표는 기존 `player_batter_stats` / `player_pitcher_stats` 테이블에 **동일한 EAV 패턴**으로 저장된다.

```sql
-- 예: 2025시즌 특정 타자의 세이버메트릭스 조회
SELECT category, value
FROM player_batter_stats
WHERE player_id = 12345
  AND season = 2025
  AND category IN ('BABIP', 'ISO', 'K%', 'BB%', 'wOBA', 'AB/HR', 'PA/K');
```

- `category` 컬럼으로 기본 지표 / 파생 지표 구분
- `saveBatterStats()` / `savePitcherStats()` 에서 기존 레코드 존재 시 UPDATE, 없으면 INSERT (upsert)
- `ranking` 필드는 null로 저장 (파생 지표는 별도 랭킹 미산정)

## 계산 시 주의사항

1. **분모 0 방지**: 모든 계산 메서드에서 분모 ≤ 0이면 0.0 반환
2. **조건부 저장**: AB/HR은 HR > 0, PA/K는 SO > 0, LOB%는 계산값 > 0일 때만 저장
3. **LOB% 클램프**: 0 ~ 1 범위로 제한 (`Math.max(0, Math.min(1, value))`)
4. **position null 처리**: DB에 position이 null인 경우 `.filter(p -> p != null && !p.isBlank())` 후 기본값 사용 (타자: "DH", 투수: "P")
5. **wOBA 단타 역산**: 1B(단타) = H - 2B - 3B - HR
6. **wOBA 가중치**: FanGraphs 2024 기본 가중치 사용 (MLB 기준이므로 KBO와 소폭 차이 가능)

## 향후 확장 가능 지표

현재 데이터로는 계산이 어렵지만, 추가 데이터 확보 시 구현 가능한 지표:

| 지표 | 필요 데이터 | 설명 |
|------|------------|------|
| wRC+ | 리그 평균 wOBA, 파크팩터 | 파크팩터 보정 득점 생산력 |
| FIP | ER, IP, HR, BB, HBP, SO + FIP 상수 | 수비 무관 투구 평가 (리그 ERA 필요) |
| xFIP | FIP + 리그 HR/FB% | 플라이볼 피홈런 보정 FIP |
| WAR | 다수 파생 지표 조합 | 대체 선수 대비 승리 기여도 |
| WHIP | (BB + H) / IP | 이닝당 출루 허용 (크롤링 데이터로 계산 가능) |
| OPS+ | OBP, SLG + 리그 평균, 파크팩터 | 파크팩터 보정 OPS |
